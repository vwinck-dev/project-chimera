Privacy Policy

Effective Date: November 10, 2025

Project Chimera (source code) is an open-source Discord bot developed by independent contributors. At this stage, Chimera does not collect, store, or process any user data.

Data Collection

No personal information is gathered from users, servers, or messages. All operations happen locally or transiently through the Discord API, without persistent logging or external storage.

Future Updates

Future releases may include optional features that store limited data like preferences or command logs. This policy will always be updated and versioned in the repository.

Transparency

Being open-source, all data handling can be audited directly. Contributions and suggestions to improve privacy are welcome.

Contact

Questions? Open an issue on the GitHub Issues page.
