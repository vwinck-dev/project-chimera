Política de Privacidade

Data de Vigência: 10 de novembro de 2025

Project Chimera (código-fonte) é um bot de Discord open-source desenvolvido por colaboradores independentes. No momento, o Chimera não coleta, armazena ou processa dados de usuários.

Coleta de Dados

Nenhuma informação pessoal é coletada de usuários, servidores ou mensagens. Todas as operações ocorrem localmente ou de forma transitória via API do Discord, sem registros persistentes ou armazenamento externo.

Atualizações Futuras

Futuras versões podem incluir funcionalidades opcionais que armazenem dados limitados, como preferências ou logs de comandos. Esta política será sempre atualizada e versionada no repositório.

Transparência

Por ser open-source, toda lógica de tratamento de dados pode ser auditada diretamente. Contribuições e sugestões para melhorar a privacidade são bem-vindas.

Contato

Dúvidas ou sugestões? Abra uma issue na página de Issues do GitHub.
