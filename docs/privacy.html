Privacy Policy

Effective Date: November 10, 2025

Project Chimera (source code) is an open-source Discord bot developed and maintained by independent contributors. At the current stage of development, Chimera does not collect, store, or process any user data.

Data Collection

As of now, no personal or identifiable information is gathered from users, servers, or messages. All operations are handled locally or through the Discord API in a transient manner — no persistent logging or external storage is involved.

Future Updates

In future releases, Chimera may introduce optional features that require limited data storage (for example, server configurations, user preferences, or command logs). Should this occur, this policy will be updated accordingly and clearly versioned in the repository.

Transparency

Because Chimera is open source, all data-handling logic can be reviewed directly in the public repository. Contributors and users are encouraged to audit and suggest improvements regarding privacy practices.

Contact

For questions, concerns, or contributions related to privacy, please open an issue on the GitHub Issues page.
